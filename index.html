FileURLBookmark

Select. Ask AI. Done.

파일과 URL을 트리 구조로 정리하고, 항목을 선택하면 AI가 즉시 요약·분석·기록까지 해주는 올인원 북마크 데스크톱 도구입니다. 아래 이미지는 실제 동작 화면이며, 클릭하면 크게 볼 수 있습니다.

Why

왜 만들었는가

⚠ 기존의 불편함

  • 파일, URL, 메모가 각기 다른 앱에 흩어져 관리 불가
  • 뉴스 기사나 문서를 요약하려면 매번 복사-붙여넣기 반복
  • AI에게 물어본 결과를 따로 저장할 곳이 없음
  • 북마크가 늘어날수록 "어디에 뒀더라?" 검색 지옥
➡

✅ FileURLBookmark의 해결

  • 파일/URL/메모를 하나의 트리에서 계층적으로 관리
  • 항목 선택 후 버튼 한 번이면 AI가 본문을 읽고 요약
  • AI 대화 이력이 항목별로 DB에 자동 보존
  • 트리 검색 + 체크 관리로 원하는 항목 즉시 접근
Features

핵심 차별점

다른 북마크 도구에는 없는, FileURLBookmark만의 기능

📡

Jina Reader 연동

URL을 선택하면 Jina Reader가 웹 페이지 본문을 자동 추출하여 AI에 전달합니다. 복사-붙여넣기 불필요.

📋

Clipboard 즉시 분석

기사를 드래그 복사한 뒤 Clipboard 모드를 선택하면, 클립보드 내용이 바로 AI에 전달됩니다.

💬

AI 대화 이력 DB 보존

요약·분석 결과가 북마크 항목별로 SQLite DB에 자동 저장됩니다. 언제든 원본 복원 가능.

🎯

프리셋 시스템 프롬프트

요약/분석/기타 3종 프리셋으로 일관된 품질의 AI 응답을 받습니다. 외부 txt 파일로 커스터마이징 가능.

📂

파일 본문 직접 전송

텍스트 파일을 북마크하면 파일 내용을 읽어서 AI에 그대로 전달합니다. 코드 리뷰에도 활용 가능.

🔄

멀티 모델 런타임 전환

gpt-4o, gpt-4.1, gpt-5.1 등 모델을 재시작 없이 실시간 전환. 배열 설정으로 모델 추가도 한 줄.

Quick Start

빠른 시작 가이드

1 다운로드 및 실행

FileBookmark.ahk 파일을 다운로드하고 AutoHotkey v1이 설치된 환경에서 실행합니다.

2 API 키 등록

OpenAI API 키를 Windows 환경변수에 등록합니다.

OPENAI_API_KEY = sk-...

3 북마크 추가 후 AI 활용

파일이나 URL을 트리에 추가하고, [AI 도우미] 버튼으로 요약/분석을 시작하세요.

FileBookmark_help.txt

[Basic Usage]
- Select an item in the tree and edit file name/path/description in the right panel.
- [Save] commits values to memory only. DB write happens on [Save and Close].

[Tree Buttons]
- Add R: Create a Root item (Level 1).
- Add S: Create an item at the same level as current selection.
- Add C: Create a child item under current selection (up to Level 3).
- Delete: Delete selected item.
- DelView: Delete multiple checked items.

[Bottom Buttons]
- Window Size: Set main window height and save to INI.
- Sort A-Z: Sort tree alphabetically while preserving hierarchy.
- Save and Close: Save bookmarks to DB and close.
- No Save Close: Close without saving bookmark changes.

[Quick Open]
- Open ChatGPT/Gemini/Claude instantly.
- Open Selected File/URL / Copy Selected Path supported.

[AI Helper]
- Click [AI 도우미] under Description to open AI chat.
- [요약]/[분석]/[기타] send current item context.
- [저장] saves edited AI chat to DB.
- [원본] reloads AI chat from DB.
- [닫기]/X closes AI window (asks when unsaved edits exist).

OpenAI_API_Setting.txt

[OpenAI API 설정 안내]

1) OpenAI API 키 발급
- https://platform.openai.com/ 로그인
- API Keys 페이지에서 "+ Create new secret key" 클릭
- 생성된 sk-... 키를 복사

2) 결제/크레딧 준비
- ChatGPT Plus/Pro 구독과 OpenAI API 결제는 별도입니다.
- API 사용 전 결제 수단 등록 + API 크레딧 충전이 필요합니다.
- OpenAI 도움말 기준 선결제 최소 구매 금액은 보통 USD 5 입니다.
- 정책은 바뀔 수 있으니 결제 직전 Billing 화면을 확인하세요.

3) Windows 환경변수 등록 (권장)
- 시작 메뉴에서 "시스템 환경 변수 편집" 검색
- "환경 변수" 버튼 클릭
- 사용자 변수에서 새로 만들기
  변수 이름: OPENAI_API_KEY
  변수 값: 발급받은 sk-... 키
- 확인 후 프로그램 재실행

4) 점검 항목
- 변수 이름 철자: OPENAI_API_KEY
- 키 앞뒤 공백/줄바꿈 제거
- 키 만료/삭제 여부 확인
- OpenAI API 결제/한도 상태 확인
